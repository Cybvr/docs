
# Getting Started with Juju

Juju is your startup's design companion that turns prompts into professional web and graphic designs. This guide will help you get started with using Juju effectively.

## Quick Start

1. Sign up for a Juju account
2. Access the dashboard 
3. Start with the prompt box in the home page
4. Generate your first design

## Features

- **AI-Powered Design Generation**: Turn text prompts into working designs
- **Real-time Editing**: Edit and customize generated designs
- **Code Export**: Export your designs as code
- **Community Access**: Browse and learn from community designs

## Basic Usage

1. Enter your design concept in the prompt box
2. Click generate to create your design
3. Edit and customize as needed
4. Export or save your design

## Initial Setup

1. Configure your preferences
2. Set up your knowledge base
3. Configure your design preferences
4. Start creating!
