
# Troubleshooting Guide

## Common Issues

### Design Generation Issues
1. **Prompt Not Working**
   - Check token balance
   - Verify prompt clarity
   - Try refreshing the page

### Account Issues
1. **Login Problems**
   - Clear browser cache
   - Reset password if needed
   - Contact support

### Export Issues
1. **Code Export Fails**
   - Check file format
   - Verify export settings
   - Try different browser

## Getting Help

1. Check FAQ section
2. Contact support
3. Visit community forums
